Данное приложение служит для отображения событий, мероприятий, новостей, взятых с kudago. 
Архитектура приложения - MVP
В приложении были использованы следующие технологии:
  - Java 8
  - Google Map API
  - RecyclerView
  - CardView
  - Retrofit 2
  - GSON
  - RxJava 2
  - Dagger 2
  - Glid
